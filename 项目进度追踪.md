     ├── search_textbook         → ChromaDB

     ├── calculate               → Python math
     ├── generate_quiz           → LLM
     ├── evaluate_answer         → LLM
     └── list_available_textbooks → 反幻觉

## Bug 修复记录

| 问题 | 根因 | 修复 |
|------|------|------|
| 回答循环重复 | Teacher 提示词"引导性问题"→ Supervisor 重路由 | 改提示词 + 简化架构 |
| Quiz 返回空 | agents.json 无效模型名 `qwen3.6-plus` | 改为 `qwen-plus` |
| 会话历史 0 条 | saveSession 未传 title/message_count | 发送真实数据 |
| 点击历史不切换 | onMounted 只触发一次 | watch + immediate |
| 模型身份冲突 | 5 文件拼接 | 合并为 teacher.md |
| LLM 编造教材列表 | 无数据来源约束 | list_available_textbooks 工具 |
| DeepSeek 不兼容 | thinking 模式 | extra_body 关闭 |
