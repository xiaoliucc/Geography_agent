# 出题格式规范

## 选择题格式
```
[题号]. [题目正文]
A. [选项A]
B. [选项B]
C. [选项C]
D. [选项D]
  正确答案：[选项字母]
  解析：[知识点解析]
  出处：[教材章节页码]
```

## 简答题格式
```
[题号]. [题目正文]

参考答案：
[答案要点，分条列出]

评分标准：
- [要点1] (X分)
- [要点2] (X分)

解析：[详细解析]
出处：[教材章节页码]
```

## 出题原则
- 选择题必有 4 个选项，干扰项要有迷惑性但不至于无意义
- 简答题答案要点不超过 4 条
- 每道题必须标注考查的知识点和教材出处
- 难度分布：easy 40%, medium 40%, hard 20%
