# 项目代码索引（v3 精简版）

> 单 Teacher Agent + 5 工具。每个文件/函数/类一两句说明。

---

## backend/

### main.py — FastAPI 入口
- `async root()` — 返回 API 根路径信息

### config/settings.py — LLM 工厂
- `class LLMFactory` — 多 LLM 提供商工厂（DashScope / DeepSeek），实例自动缓存
  - `create(provider, model, temperature, max_tokens)` — 创建 ChatOpenAI 实例
  - `create_for_agent(agent_type, **kwargs)` — 从 agents.json 读取配置创建 LLM

### config/agents.json — Agent 模型配置
- `teacher` — 教学 Agent 用 qwen-plus
- `quiz` — 出题/批改用 qwen-plus

### config/prompts/
- `personas/teacher.md` — "地老师"人设：教学风格、能力范围、来源标识规则、回答长度
- `skills/quiz_format.md` — 出题格式规范：选择题/简答题标准模板

---

### db/__init__.py — 向量数据库
- `get_all_vector_dbs()` — 扫描 materials/processed/ 下所有 ChromaDB
- `async embed_text(text)` — DashScope text-embedding-v4 嵌入，MD5 缓存
- `async search_textbook(embedding, top_k, chapter_id, textbook_filter)` — 并行搜索 ChromaDB，按距离排序

---

### agent/__init__.py — L1 对话记忆
- `get_checkpointer_db()` — 返回 conversations.db 路径，确保目录存在

### agent/agents/__init__.py — 单 Teacher Agent
- `_load_persona(name)` — 加载 Layer 3 人设
- `_load_skills(*names)` — 加载 Layer 2 规则
- `create_teacher_agent()` — 创建全能 Teacher，包含 5 个工具

### agent/tools/
- `textbook_search.py` — `async search_textbook(query, top_k, chapter_id)` ChromaDB 教材检索
- `web_search.py` — `async web_search(query, max_results)` Tavily 网络搜索
- `calculator.py` — `calculate(expression, calc_type)` 安全沙箱地理计算
- `quiz_tools.py` — `async generate_quiz(...)` / `async evaluate_answer(...)` 出题与批改

### agent/memory/
- `models.py` — SQL 建表语句（user_profiles / conversation_summaries / learning_progress）
- `user_profile.py` — L2 CRUD
  - `async get_db()` — 获取连接 + 建表
  - `async get_or_create_user(user_id)` — 获取/创建用户
  - `async update_user_profile(user_id, **fields)` — 更新年级/教材
  - `async save_session_summary(...)` — 保存会话摘要
  - `async list_sessions(user_id, limit, offset)` — 历史会话列表
  - `async record_interaction(...)` — L3 互动记录（预留）

---

### api_v2/chat.py — 主对话 SSE 端点
- `class ChatRequest(BaseModel)` — 请求体：message、user_id、session_id、mode
- `_sse(event, data)` — 格式化 SSE 字符串
- `_safe_serialize(obj)` — 安全 JSON 序列化
- `async chat(request)` — 创建 Teacher Agent + checkpointer → SSE 推送 tool_call/chunk/done

### api_v2/quiz.py — 出题端点
- `class QuizGenerateRequest(BaseModel)` — 出题请求：topic、question_count、difficulty、question_types
- `class QuizEvaluateRequest(BaseModel)` — 批改请求：question、student_answer
- `async generate(request)` — 调用 generate_quiz 工具
- `async evaluate(request)` — 调用 evaluate_answer 工具

### api_v2/user.py — 用户端点
- `class UpdateProfileRequest(BaseModel)` — 更新档案请求
- `class SaveSessionRequest(BaseModel)` — 保存会话请求
- `async get_profile(user_id)` — 获取/创建用户档案
- `async update_profile(request)` — 更新年级/教材
- `async list_sessions(...)` — 历史会话列表
- `async save_session(request)` — 保存会话摘要

### api_v2/health.py — 健康检查
- `async health()` — 各组件连通状态
- `async list_textbooks()` — 已索引教材列表

### api_v2/exceptions.py — 异常处理
- `class APIError(Exception)` — 统一 API 异常（code + message + http_status）
- `api_error_handler(func)` — 装饰器：异常 → JSON 错误响应

### api_v2/middleware.py — 中间件
- `class RequestIDMiddleware` — 请求追踪 ID 注入
- `class LoggingMiddleware` — 请求日志（method + path + status + ms）

---

### pipeline/ — 数据处理管道（6 文件）

### pipeline/__init__.py — Qwen-VL 共用客户端
- `get_client()` — 获取/缓存 OpenAI 客户端
- `ask(prompt, image_b64, max_tokens)` — 调用 Qwen-VL，可选传图

### pipeline/gui.py — tkinter 图形导入工具
- `class ImportGUI` — 扫描、预览、一键导入（4 种模式 + 5 步进度条）
  - `_scan()` — 后台扫描 textbooks 目录，标注 PDF 类型
  - `_import(pdf_path, pub, book, mode)` — 核心导入：提取→全页图→分块→嵌入→ChromaDB
  - `_extract(pdf_path, pe, mode)` — 按模式分流（增强/电子/OCR/混合）

### pipeline/digital.py — 电子增强管道
- `extract_page_images(page)` — 提取嵌入位图 + 检测非文字区域（矢量图/表）
- `match_captions_to_images(blocks, images, page_num)` — 图标题与图片按 y 坐标关联
- `describe_image(image_bytes, caption)` — Qwen-VL 描述图片内容
- `process(pdf_path, output_dir)` — 处理 PDF → pages/ + markdown/ + figures/
- `build_chunks(pages, figures, output_dir)` — 正文 + 图描述混合分块

### pipeline/ocr.py — Qwen-VL OCR 管道
- `render_page(pdf_path, page_num, scale)` — PDF 单页渲染为 PNG base64
- `ocr_page(image_b64)` — Qwen-VL 识别 → 结构化 JSON
- `page_json_to_markdown(page_data, page_num)` — JSON → Markdown
- `process(pdf_path, output_dir, start, end)` — 扫描件批量 OCR

### pipeline/full_pages.py — 全页图片
- `generate(pdf_path, output_dir, scale)` — 渲染全页 PNG（前端 PDF 查看器用）

### pipeline/classifier.py — PDF 类型检测
- `classify(pdf_path)` — 检测类型（born_digital/mixed/scanned）
- `scan(textbook_dir)` — 批量扫描目录

---

### agent/tools/system_tools.py — 系统信息工具
- `list_available_textbooks()` — 返回实际已索引的教材列表，防止 LLM 编造教材

### agent/memory/knowledge_points.py — 知识点匹配
- `match_knowledge_points(topic)` — 将话题文本匹配到 66 个课标知识点
- `list_all_points()` — 列出全部知识点（必修一 + 必修二）

## frontend/

### src/main.ts — Vue 入口
- `createApp(App)` — 注册 Pinia + Router，挂载

### src/App.vue — 根布局

- `/quiz` → QuizGenerator

- `UserProfile` / `ConversationSummary` — 用户数据

- `sendMessage(text)` — 发送消息 + SSE 流
- `toggleMessageExpand(id)` — 折叠/展开工具调用
- `switchSidebarTab(tab)` / `setMode(mode)` — UI 状态

### src/components/chat/ChatContainer.vue — 主聊天界面

### src/components/chat/ToolCallDisplay.vue — 工具调用卡片

### src/components/sidebar/Sidebar.vue — 双面板侧边栏
- 快捷导航（对话/做题/报告）→ Tab 切换（会话/学习）
- `newChat()` / `onMounted()`

### src/components/quiz/QuizGenerator.vue — 出题页面
- 表单 → 答题区 → 结果展示 三态切换
- `generate()` / `submit()` / `reset()` / `renderMarkdown(text)`

### src/components/dashboard/LearningDashboard.vue — 学习报告

- `onMounted()` 加载数据
